# 📇 Contact Manager (Full Stack MERN App)

A modern full-stack Contact Manager application built using **MongoDB, Express, React, and Node.js (MERN stack)**.

This app allows users to create, view, update, search, and delete contacts through a clean and professional interface.

---

## 🚀 Features

- ✅ Add new contacts
- ✅ Edit existing contacts
- ✅ Delete contacts
- ✅ Search contacts instantly
- ✅ Persistent data using MongoDB Atlas
- ✅ Form validation with user-friendly feedback
- ✅ Responsive modern UI
- ✅ Loading states and empty states
- ✅ Clean UX and professional design

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Axios
- CSS (Custom modern design)

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
